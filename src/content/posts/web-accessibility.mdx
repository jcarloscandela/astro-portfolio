# Web Accessibility in Modern Applications

## Semantic HTML

Using semantic HTML elements is the foundation of web accessibility:

```html
<!-- Better accessibility -->
<article>
  <header>
    <h1>Main Title</h1>
    <p>Description</p>
  </header>
  <main>
    <section>
      <h2>Section Title</h2>
      <!-- Content -->
    </section>
  </main>
</article>
```

## ARIA Attributes

ARIA attributes enhance accessibility when HTML semantics aren't enough:

- Use aria-label for elements without visible text
- Implement aria-live for dynamic content
- Add aria-expanded for toggleable components

## Keyboard Navigation

Ensure your application is fully navigable via keyboard:

- Logical tab order
- Visible focus indicators
- Keyboard shortcuts for common actions

Remember: accessibility benefits everyone, not just users with disabilities!
